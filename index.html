COMMANDES POUR LE RACCOURCI CLAVIER

"afin de faciliter le codage,je vous propose quelques raccourcis claviers sur le tableau ci-dessous"

ACTIONS RACCOURCI CLAVIERS
tue tous les programmes alt+sys+I
redemmarage brutal de l ordi alt+sys+B
arret butal de l ordi alt+sys+O
faire apparaitre le menu de la fenetre alt+epace
maximiser une fenetre alt+f10
menu application alt+f1
lancer une commande alt+f2
deplacer la fenetre alt+f7
synchronisation des disques alt+sys+S
stoppe les programmes gentiments alt+sys+E
zoom ctrl+molette souris
changer le bureau ctrl+ ← ou →
redimensionner une fenetre alt+clic molette souris
